# AI-Teks-Sentimen
This is an AI that can detect sentiment from the text we input, it can be useful for customer service, filtering comments and others.


how to use ?

First of all, you can open the file with vs code that has installed the python library, then after you download and enter it into vs code, the next step is to run the "app.py" program, this is useful for connecting AI with UI, after that you can run the "app.jsx" file in the "src" folder, and if both are running then you only need to open the website that is running on localhost on your device
